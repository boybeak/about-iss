# ✨ 主要功能

## 两种显示风格

| 风格 | 描述 |
|------|------|
| **Simple（简洁）** | 仅在当前输入法为指定输入法时显示一个小圆点指示器 |
| **Normal（标准）** | 显示当前输入法的首字母，并在 Caps Lock 启用时显示锁定图标 |

## 个性化配置

- **全局透明度**：调整指示器的整体透明度
- **空闲延迟**：设置鼠标停止移动后指示器消失的延迟时间
- **空闲透明度**：设置空闲状态下指示器的透明度
- **指定输入法**：在 Simple 模式下选择要监控的输入法
- **深色模式适配**：自动跟随系统主题，也可手动切换

## 菜单栏功能

- **Caps Lock 开关**：快速切换大写锁定状态
- **显示风格切换**：选择 Simple 或 Normal 模式
- **在 Dock 中隐藏**：切换是否在 Dock 中显示应用图标
- **开机自启动**：设置应用是否随系统启动
- **设置**：打开详细设置界面
- **关于**：查看应用信息（跳转至本页面）
- **退出**：关闭应用

## 技术特性

- **完全本地运行**：无需网络连接，数据隐私安全
- **SwiftUI 构建**：现代化的界面设计，原生 macOS 体验
- **轻量级资源占用**：极低的内存和 CPU 占用
