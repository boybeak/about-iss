# InputSourceSimple

一款简洁的 macOS 菜单栏辅助工具，帮助用户在使用多输入法时快速识别当前所处的输入法状态。

## 产品简介

**InputSourceSimple** 是一款轻量级的 macOS 菜单栏应用，当鼠标悬停时自动显示当前输入法状态指示器。

### 核心功能

- **悬停显示**：当鼠标移动到屏幕顶部时，自动显示当前输入法状态指示器
- **Caps Lock 监控**：实时显示大写锁定键的状态
- **轻量级设计**：无 Dock 图标干扰，保持桌面整洁
- **开机自启**：支持开机自动启动，方便日常使用

### 主要特性

#### 两种显示风格

| 风格 | 描述 |
|------|------|
| **Simple（简洁）** | 仅在当前输入法为指定输入法时显示一个小圆点指示器 |
| **Normal（标准）** | 显示当前输入法的首字母，并在 Caps Lock 启用时显示锁定图标 |

#### 个性化配置

- **全局透明度**：调整指示器的整体透明度
- **空闲延迟**：设置鼠标停止移动后指示器消失的延迟时间
- **空闲透明度**：设置空闲状态下指示器的透明度
- **指定输入法**：在 Simple 模式下选择要监控的输入法

## 使用说明

### 菜单栏操作

点击菜单栏图标可展开以下选项：

- **Caps Lock 开关**：快速切换大写锁定状态
- **显示风格切换**：选择 Simple 或 Normal 模式
- **在 Dock 中隐藏**：切换是否在 Dock 中显示应用图标
- **开机自启动**：设置应用是否随系统启动
- **设置**：打开详细设置界面
- **关于**：查看应用信息
- **退出**：关闭应用

### 指示器显示

- 将鼠标移动到屏幕顶部区域，指示器将出现在鼠标光标附近
- 鼠标停止移动后，指示器会在设定的延迟时间后自动淡出
- 指示器会同时显示在所有连接的显示器上

## 系统要求

- macOS 13.0 或更高版本
- 需要辅助功能权限（用于全局监听输入法切换事件）

## 技术架构

- **UI 框架**：SwiftUI
- **核心模块**：
  - `InputMethodMonitor`：监听键盘输入法切换
  - `CapsLockMonitor`：监控大写锁定键状态
  - `IndicatorManager`：管理多显示器上的指示器窗口
  - `AppConfig`：应用配置与持久化存储
