# 📖 快速上手

## 首次使用

1. 首次启动后，应用会显示在菜单栏中（一个图标）
2. 点击菜单栏图标可展开功能菜单

## 菜单栏操作

点击菜单栏图标可展开以下选项：

- **Caps Lock 开关**：快速切换大写锁定状态
- **显示风格**：选择 **Simple（简洁）** 或 **Normal（标准）** 模式
- **在 Dock 中隐藏**：切换是否在 Dock 中显示应用图标
- **开机自启动**：设置应用是否随系统启动
- **设置**：打开详细设置界面
- **关于**：查看应用信息（跳转至本页面）
- **退出**：关闭应用

## 指示器显示

- 将鼠标移动到屏幕顶部区域，指示器将出现在鼠标光标附近
- 鼠标停止移动后，指示器会在设定的延迟时间后自动淡出
- 指示器会同时显示在所有连接的显示器上

## 设置说明

在设置中您可以：

| 设置项 | 说明 |
|-------|------|
| **显示风格** | 选择 Simple 或 Normal 模式 |
| **全局透明度** | 调整指示器的整体透明度 (0-100%) |
| **空闲延迟** | 鼠标停止移动后指示器消失的延迟时间 (秒) |
| **空闲透明度** | 空闲状态下指示器的透明度 (0-100%) |
| **指定输入法** | 在 Simple 模式下选择要监控的输入法 |

> 💡 提示：建议开启「开机自启动」，应用会驻留在后台菜单栏，不占用 Dock 图标位。
